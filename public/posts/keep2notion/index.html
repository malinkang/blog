将运动数据自动同步到Notion

Apr 15, 2025
1 minute read

最近在健身房发现人比刚过完年那段时间人少多了。自己也变得越来越堕落了，每月跑100公里的目标两个月都没有完成。刚好最近不忙，重新把运动数据同步到Notion这个项目的模板优化下。

一年前，发布了将跑步数据自动同步到Notion。上线之后遇到的问的最多的三个问题就是：

  1. 能否支持其他运动类型

  2. 能否支持苹果运动、小米运动、华为运动以及其他运动。

  3. 能否将体重数据同步到Notion

对于第一个问题，目前已经支持所有的运动类型。跑步有3中类型：跑步、室内跑步和室外跑步，跑步的数据包含加了室内跑步和室外跑步。同理，步行和骑行也有有三种类型。

每个运动类型都有热力图和图标。我们可以点击某个运动类型打开就可以看到热力图和图表。图表可以根据自己的需求修改，不需要的话可以删除，不会影响同步。注意Notion免费用户只能创建一个图表，需要显示更多图表，可以联系我付费升级教育Plus，十元永久使用。

对于第二个问题，Keep支持将苹果健康、小米运动、华为运动、佳明、高驰等运动数据同步到Notion。没有使用过Keep的同学可以先下载Keep App，然后点击我的、点击右上角的手表图标，添加应用服务，将自己的运动数据同步到Keep，然后再自动同步到Notion。

第三个问题，目前也支持将体重数据同步到Notion，搭配上Notion的图标功能，让体重数据可视化，更加能激励自己。

此外还增加了我的设备、可以在Keep中添加设备，然后同步到Notion中。

以上就是这次更新的全部内容，可以查看我已经同步的效果，了解更多信息。https://malinkang.notion.site/1ca86019c92c8103bf6acbd5007d7483?pvs=4

使用

Fork工程

打开Keep2Notion，点击右上角的Fork（顺便点个star谢谢）

复制Notion模板

  1. 浏览器打开：https://api.notion.com/v1/oauth/authorize?client_id=89f93a9e-a400-45f8-98dc-051c8f6488d3&response_type=code&owner=user&redirect_uri=https%3A%2F%2Fnotion-auth.malinkang.com%2Fkeep-oauth-callback

  2. 然后点击Next->Allow access

在Github的Secrets中添加变量

打开你fork的工程，点击Settings->Secrets and variables->New repository secret

依次输入如下值：

Name Secret
KEEP_MOBILE Keep手机号
KEEP_PASSWORD Keep密码
NOTION_TOKEN 第二步复制的NOTION_TOKEN
NAME 热力图名字
NOTION_PAGE 第二步复制的NOTION_PAGE

运行

打开你Fork的项目依次点击Action->Keep sync->Run workflow
